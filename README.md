Интерактивное веб-приложение с колесом удачи, системой квестов и управлением пользовательским аккаунтом. Приложение построено на современном стеке React с TypeScript и включает анимации, состояние через Redux и адаптивный дизайн.

## ✨ Основные возможности

- **🎡 Колесо удачи** - интерактивное вращающееся колесо с призами
  - Анимации с использованием Framer Motion
  - Система кулдауна между вращениями
  - Визуализация выигранных призов
- **📋 Система квестов** - ежедневные задания и бонусы
  - Прогресс-бар последовательных дней
  - Взаимодействие с бонусами (поделиться, отзыв, пригласить и т.д.)
  - Награды за выполнение квестов
- **👤 Управление аккаунтом** - просмотр и редактирование данных пользователя
  - Отображение email и пароля
  - Переключение между квестами и информацией об аккаунте
- **🎨 Современный UI/UX**
  - Адаптивный дизайн с SCSS модулями
  - SVG иконки через спрайт-систему
  - Кастомные компоненты иконок
- **📱 Полностью адаптивный** - работает на мобильных устройствах и десктопах

## 🛠 Технологический стек

- **Frontend Framework**: React 19.2.4 с TypeScript
- **Build Tool**: Vite 8.0.1
- **State Management**: Redux Toolkit 2.11.2 + React Redux 9.2.0
- **Animations**: Framer Motion 12.38.0
- **Styling**: SCSS/SASS с модулями
- **Code Quality**: ESLint 9.39.4 + TypeScript ESLint
- **SVG Handling**: vite-plugin-svg-spriter
- **React Compiler**: babel-plugin-react-compiler (экспериментальный)

### Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd vite-project
```

2. Установите зависимости:
```bash
npm install
# или
yarn install
# или
pnpm install
```

3. Запустите сервер разработки:
```bash
npm run dev
# или
yarn dev
# или
pnpm dev
```

4. Откройте браузер по адресу [http://localhost:5173](http://localhost:5173)

## 🔧 Конфигурация

### Vite
Конфигурация Vite находится в `vite.config.ts` и включает:
- Плагин React с компилятором
- Плагин Babel для React Compiler
- Плагин SVG спрайтов
- Алиасы для путей (`@/`, `@components/`, `@assets/`)

### TypeScript
Настроены два конфига:
- `tsconfig.json` - общая конфигурация
- `tsconfig.app.json` - для приложения
- `tsconfig.node.json` - для Node.js окружения

### ESLint
Конфигурация ESLint в `eslint.config.js` с поддержкой:
- TypeScript
- React Hooks
- React Refresh

## 📱 Особенности реализации

### Колесо удачи (`src/features/Wheel/`)
- Использует Framer Motion для плавных анимаций вращения
- Рассчитывает углы для каждого приза на основе данных из store
- Реализует систему кулдауна (24 часа между вращениями)
- Отображает модальное окно с выигранным призом
- Сохраняет состояние в локальное хранилище

### Система квестов (`src/features/quests/`)
- Отслеживает последовательные дни вращения колеса
- Предоставляет бонусы за социальные действия
- Визуализирует прогресс через progress bar
- Интегрируется с пользовательским store

### Управление состоянием
- **Redux Toolkit** для централизованного состояния
- **Слайсы**: `userSlice` (данные пользователя) и `prizesSlice` (призы)
- **Кастомные хуки**: `useUser`, `usePrizes`, `useAppDispatch`
- **Mock API**: имитация серверных запросов через `dataApi.ts`

### Стилизация
- **SCSS модули** для изоляции стилей компонентов
- **Глобальные переменные** в `src/shared/styles/_variables.scss`
- **Миксины** для адаптивного дизайна
- **Кастомные шрифты** (Alumni Sans, Kelly Slab)

### SVG иконки
- Все иконки собраны в единый SVG спрайт
- Компонент `CustomIcon` для удобного использования
- Поддержка кастомных размеров и цветов через props

## 📊 Данные приложения

Приложение использует mock данные из `src/shared/api/database/data.json`:
- **Пользователь**: email, пароль, время последнего вращения, бонусы
- **Призы**: название, описание, изображение, сообщение при выигрыше
